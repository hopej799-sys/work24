CS 사용 가이드 (PoC)

모니터링 피드백 구축 배경

2025년 캄보디아 해외취업 사기 사건 이후 거짓 구인광고에 대한 사회적 경각심이 높아지면서, 고용24(한국고용정보원)가 민간 취업포털을 대상으로 구인공고 모니터링 피드백 API를 제공하기 시작했습니다.

목적

고용24는 잡코리아에서 연동된 공고를 모니터링한 후, 자체 기준에 따라 게재 불가 사유를 피드백으로 안내하고 있으며, 이 피드백을 참고하여 잡코리아 내부 정책 기준으로 해당 공고를 재검토하고 처리 여부를 판단하는 것이 이 업무의 목적입니다.

사용 가능한 필터
처리 상태 (전체 / 미검토 / 검토중 / 검토완료 / 이상없음 / 게재중단) 오류구분 (전체 / 사전필터링 / 구인 모니터링) 법위반의심 Y만 법령 맵핑 내용 있는 것만 메모 있는 것만
✅ 권장 일일 업무 순서

FAQ

자주 묻는 질문

Q 조회했는데 데이터가 없어요.
해당 기간에 고용24 피드백이 없는 것입니다. 기간을 넓히거나 다음 날 다시 확인하세요.
Q 저장했는데 다음에 조회하면 처리상태가 안 보여요.
다른 기간으로 조회하면 해당 공고번호가 결과에 포함되지 않을 수 있습니다. 공고번호로 직접 검색하거나 해당 날짜로 조회하면 저장된 상태를 확인할 수 있습니다.
Q 게재중단으로 바꿨는데 실제 공고 처리는 어떻게 하나요?
이 도구는 처리 상태를 기록하는 용도입니다. 실제 공고 게재중단은 기존 잡코리아 어드민에서 별도로 처리해야 합니다.
Q 여러 명이 동시에 사용할 수 있나요?
네, 가능합니다. 저장 시 공고번호 기준으로 덮어쓰므로 마지막에 저장한 내용이 유지됩니다. 동시 작업 시 서로 다른 공고를 담당하도록 조율하세요.
Q 필터링 정책 이슈는 어떻게 처리하나요?
메모란에 이슈 내용을 기록하고 도메인 담당자에게 메일로 전달합니다. 메일 전달 후 메모에 이력도 함께 남겨주세요.
